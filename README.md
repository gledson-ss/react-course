## Types of commits accepted
 *Start with the word feat (feature).*
 - feat: `:sparkles:` new file created
 - feat: `:pencil:` update or add features
 - feat: `beetle` warn that the application has a bug
 - feat: `:hammer:` bug fix
 - feat: `:x:` delete file
